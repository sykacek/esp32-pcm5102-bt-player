## ESP32 PCM5102 Bluetooth player

How to build:


1. Clone the repo:
```
	git clone https://github.com/sykacek/esp32-pcm5102-bt-player.git
```

2. Init submodules:
```
	git submodule update --init --recursive --progress
```

3. Activate `esp-idf` and build:
```
	idf.py build
```

### NOTE
Tested with `esp-idf/release/v6.0`
